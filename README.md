# 🎓 Premium Math - Wikipedia Explorer with Games

An interactive website that combines random Wikipedia article exploration with a games discovery portal.

## Features

✨ **Random Wikipedia Articles** - Click "Random Article" to explore random topics from Wikipedia

🔍 **Search Wikipedia** - Search for any topic on Wikipedia directly from the app

🎮 **Games Portal** - Type "games" in the search bar and press Enter to see a popup with featured games

📱 **Responsive Design** - Beautiful gradient UI that works on desktop and mobile

✨ **Smooth Animations** - Fluid transitions and interactive elements

## How to Use

1. **Open `index.html`** in your web browser
2. **Random Article**: Click the "Random Article" button to load a random Wikipedia page
3. **Search**: Enter any term and click "Search" to find Wikipedia articles
4. **Games**: Type "games" and press Enter to open the games popup

## Structure

- `index.html` - Main HTML structure
- `style.css` - Beautiful styling and animations
- `script.js` - Wikipedia API integration and game popup functionality

## Technologies

- HTML5
- CSS3 (with animations)
- JavaScript (Fetch API)
- Wikipedia REST API

## About 3kh0 Assets

This project is designed to integrate with the 3kh0 game assets collection. The games popup can be enhanced to link directly to game assets.
